Tree Lamp

Pallet be a Tree again

The pallets that carried a heavy burden all through life and take a rest by now are

recreated and designed as lighting motivated by using pallets as materials for lighting.

Since it is variable lighting products, this lighting has for two asymmetry brass for three

levels of light which means branches in common and highlights the analog sensibility with Edison light bulb.

In case of series block, it uses hexahedron blocks as props without any process,
so that it could be base of lighting while remaining its vintage mood.

In case of series space, It utilizes the pallet board as a base, so that it could be recreated

as a new space in desk by putting some decoration on it or key or accessories on some branches.
